# aesophia-parser - development

## Dependencies

- make
- [node](https://nodejs.org) >= 13.12

**Install node modules dependencies**
```
$ make deps
```

## Test

### Mocha tests

```
$ make test
```

### aesophia-parser tests
```
$ make build-dev
$ make verify-explicit-stdlib
$ make verify-valid-test-contracts 
```
